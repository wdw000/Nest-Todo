
### Create .env
```
DATABASE_HOST = YOUR DATABASE IP
DATABASE_PASSWORD = YOUR DATABASE PW
DATABASE_USER = YOUR DATABASE USER NAME
DATABASE_NAME = YOUR DATABASE NAME
DATABASE_PORT = YOUR DATABASE PORT NUMBER

LISTEN_PORT = YOUR LISTEN PORT NUMBER
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
